# 📖 Insights into the Quran — نظرات في القرآن

**بِسْمِ اللَّهِ الرَّحْمَنِ الرَّحِيمِ**

Interactive trilingual (AR/EN/FR) educational web app based on the book **نظرات في القرآن** (Insights into the Quran) by Sheikh Mohammed al-Ghazali (1917-1996).

## Features

- 20 insights and reflections on the Quran
- Miracles section: literary, scientific, legislative, psychological
- Daily Quranic habit tracker with streaks
- Knowledge quiz
- Trilingual: Arabic, English, French with RTL/LTR support
- 3 unique themes with localStorage persistence
- Narrator with karaoke, sleep timer, and lock screen controls
- Responsive design + print styles
- PWA-ready with manifest.json

## Tech

Pure HTML + CSS + JS. No frameworks, no dependencies. Just open `index.html`.

## Disclaimer

I am not a scholar. This is a humble effort by a Muslim parent to share Sheikh al-Ghazali's Quranic reflections interactively. Content derived from the book and trusted Islamic sources. This is not a fatwa.
